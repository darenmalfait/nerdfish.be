Uptime Kuma is a self-hosted monitoring tool like "Uptime Robot". Installing it on a raspberry pi can easily be done by installing it using docker.

![uptime kuma](http://res.cloudinary.com/darenmalfait/image/upload/v1668519428/68747470733a2f2f757074696d652e6b756d612e7065742f696d672f6461726b2e6a7067_rgm89c.jpg "uptime kuma")

## Create a volume

```powershell
docker volume create uptime-kuma
```

## Start the container

```powershell
docker run -d --restart=always -p 3001:3001 -v uptime-kuma:/app/data --name uptime-kuma louislam/uptime-kuma:1
```

Browse to [http://localhost:3001](http://localhost:3001/) after starting.
