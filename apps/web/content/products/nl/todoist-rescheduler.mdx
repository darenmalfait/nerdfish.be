Ik heb Todoist Rescheduler gebouwd om een persoonlijk probleem op te lossen: de stress van het zien van verlopen taken in Todoist. In plaats van taken te laten opstapelen als verlopen, verplaatst deze GitHub Action ze automatisch naar de taken van vandaag, wat zorgt voor een meer vergevingsgezinde en productieve workflow.

## Wat het doet

- **Automatische Taakherplanning**: Verplaatst verlopen taken naar vandaag in plaats van ze als verlopen te markeren
- **GitHub Action Integratie**: Draait automatisch volgens een door jou ingesteld schema
- **Zero Configuratie**: Werkt direct met alleen je Todoist API token
- **Aanpasbaar**: Optioneel filteren welke taken worden herpland
- **Gratis te Gebruiken**: Open source en draait op GitHub's gratis tier

## Hoe het werkt

1. Stel de GitHub Action in met je Todoist API token
2. Configureer het schema (bijv. dagelijks om middernacht)
3. De action vindt alle verlopen taken
4. Verplaatst ze automatisch naar de taken van vandaag
5. Houdt je Todoist schoon en stressvrij

Gebouwd met TypeScript en GitHub Actions, helpt deze tool bij het behouden van een meer vergevingsgezind taakbeheersysteem. Het is open source en gratis te gebruiken.
